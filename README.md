# Sobre o projeto:

Este projeto consiste em um site de serviços de contabilidade desenvolvido para apresentação e captação de clientes. A plataforma foi criada com o objetivo de permitir que o usuário descreva sua necessidade ou problema contábil e solicite um orçamento de forma simples e rápida.

A proposta é que, após o envio da solicitação, o cliente receba um retorno com o orçamento por e-mail ou WhatsApp. No momento, as funcionalidades de envio automático ainda estão em desenvolvimento, mas a estrutura do site já está preparada para essa integração futura.

# O site foi desenvolvido como um projeto prático, com foco em: 

- Apresentação dos serviços contábeis
- Captação de leads
- Interface simples e intuitiva
- Estrutura preparada para automação de atendimento

# Tecnologias utilizadas

- HTML5 – Estrutura e organização das páginas
- CSS3 – Estilização e layout do site
- JavaScript – Interações e validações do formulário (em desenvolvimento)
